Movies API

This is a REST API using Node.js

Objective

Build a server-side component of a movie web app.

This web app will give users access to info about action movies such as their directors and sub-genres.

Users will also be able to sign up on the site and create a favorites list.

Business Logic URL Endpoint HTTP Method Request Body Response Body
Get a list of all movies /movies GET None JSON object holding data about all the movies
Get all info (description, director, and image URL) on a movie by title /movies/:Title GET None JSON object holding data for movie that was requested
Get info about movie director by name /movies/director/:Name GET None JSON object info about director such as a short bio and birth year
Allow new users to register /users POST JSON object containing username, password, and email JSON object containing data of the user that was just added
Allow users to update username /users/:username PUT JSON object with updated info JSON object with updated info
Allow users to add movies to their favorite list /users/:Username/movies/:Title POST None A text saying that the movie has been added to favorites
Allow users to remove movies from their favorites /users/:Username/movies/:Titler DELETE None A text saying that the movie has been deleted from their favorites
Allow users to deactivate their account /users/:Username DELETED None A text stating that the user email/account has been removedand deactivated
